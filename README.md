# تحلیل رقبا و الگوریتم اینستاگرام

این مخزن شامل یک ابزار ساده برای تحلیل رقبا در اینستاگرام و امتیازدهی محتوا بر اساس اصول شناخته‌شده از الگوریتم اینستاگرام است. این ابزار نیاز به دسترسی به اینترنت ندارد و با داده‌های CSV که از قبل در اختیار دارید کار می‌کند.

ویژگی‌ها:
- محاسبه نرخ تعامل (Engagement Rate) و امتیاز هر پست با وزن‌دهی به لایک، کامنت، ذخیره و اشتراک‌گذاری
- در نظر گرفتن تازگی (Recency) با تابع کاهش نمایی (نیمه‌عمر پیش‌فرض ۷ روز)
- وزن‌دهی به نوع محتوا (Reels/Carousel/Photo/Video)
- بهینه‌بودن طول کپشن و تعداد هشتگ‌ها
- تحلیل سطح اکانت: میانگین امتیاز، نرخ تعامل، کَدِنس انتشار (پست/هفته)، نوع محتوای برتر، بهترین بازه‌ی زمانی انتشار و برترین پست‌ها
- تولید گزارش Markdown

پیش‌نیازها:
- Python 3.8+
- نیازی به نصب پکیج‌های خارجی (مثل pandas) نیست.

ساختار داده ورودی (CSV):

ستون‌های زیر پشتیبانی می‌شوند. ستون‌های ستاره‌دار ضروری‌اند:
- username* (نام کاربری)
- post_id* (شناسه پست)
- timestamp* (ISO 8601 مثل 2024-08-12T14:05:00)
- content_type* (یکی از: reel, carousel, photo, video)
- likes, comments, shares, saves (اعداد صحیح، پیش‌فرض 0)
- followers* (تعداد فالوورها در زمان پست)
- impressions, reach, video_views, watch_time_seconds (اختیاری)
- caption (اختیاری)
- hashtags_count (اختیاری؛ در صورت نبود، از کپشن استخراج می‌شود)

نمونه داده در مسیر sample_data/competitors_posts.csv قرار دارد.

نصب و اجرا:

- اجرای تحلیل و ساخت گزارش:

```
python -m ig_competitor_analysis --input sample_data/competitors_posts.csv --output reports/report.md
```

در پایان فایل گزارش در مسیر reports/report.md ایجاد می‌شود.

نکته: برای محاسبه وزن تازگی، ابزار به‌صورت پیش‌فرض «آخرین زمان پست در دیتاست» را به‌عنوان اکنون (now) در نظر می‌گیرد تا دیتاست‌های قدیمی دچار صفرشدن امتیاز نشوند.

منطق امتیازدهی (خلاصه):
- امتیاز پایه تعامل = likes + 2*comments + 3*(shares + saves)
- نرخ تعامل هر پست = امتیاز پایه / followers
- وزن تازگی = exp(-ln(2) * age_days / 7)
- وزن نوع محتوا: reel=1.2، carousel=1.1، photo=1.0، video=1.05
- وزن کپشن: بهترین محدوده 100 تا 200 کاراکتر
- وزن هشتگ: بهترین محدوده 3 تا 5 هشتگ
- امتیاز نهایی پست = نرخ تعامل × وزن تازگی × وزن نوع محتوا × وزن کپشن × وزن هشتگ
